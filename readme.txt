========================================================================
SAPG MLM Configuration Paths Readme
========================================================================

All system configurations and rules are stored in file-based settings to maintain speed, security, and version control.

1. Package Configuration Settings:
   - Path: c:/xampp/htdocs/autopool/libs/payouts.php
   - Variable: $PACKAGE_CONFIG (Array containing pricing, payouts, and referral requirements)

2. Environment Config Settings:
   - Path: c:/xampp/htdocs/autopool/.env
   - Details: Database connection settings, site name, logo path, SMTP settings.
